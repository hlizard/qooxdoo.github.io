qooxdoo 3.5.1 (2014-06-05)

Summary

This is a regular patch release, which includes some important bug fixes over the previous release. It is fully backwards-compatible to qooxdoo 3.5. Nothing needs to be changed in the JavaScript code of your existing apps if they are based on the qooxdoo 3.5 release.

Bugfixes & Enhancements

See the corresponding bugzilla query for the issues resolved for qooxdoo 3.5.1.
