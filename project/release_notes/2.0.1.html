qooxdoo 2.0.1 (2012-07-03)

Summary

This is a regular patch release, which includes some important bug fixes but no feature enhancements over the previous release. It is fully backwards compatible to qooxdoo 2.0. Nothing needs to be changed in the JavaScript code of your existing apps if they are based on the previous 2.0 release.

Bugfixes

See the corresponding bugzilla query for the issues resolved for qooxdoo 2.0.1.

Core

  • Minor fix in the application routing.

qx.Website

  • Fixed build optimization which had removed some public methods.
  • The library file names now contain the explicit version number, e.g. q-2.0.1.min.js
  • Corrected some API documentation.
  • The SDK now contains a pre-built qx.Website API viewer.
  • Added hints for extending and overriding to the API viewer.

qx.Desktop

  • Added missing migration entry for migrating to 2.0.1.
  • Fixed broken appearance of error tooltips in the Modern theme.

qx.Server

  • The file names now contain the version, e.g. qx-oo-2.0.1.js
  • Runtime checks are now included.

Applications

  • Feature Configuration Editor: Fixed calculation of duplicate entry detection.
