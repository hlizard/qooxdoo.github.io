qooxdoo 2.0.4 (2013-02-05)

Summary

This is a patch release, which includes one important bug fix but no feature enhancements over the previous release. It is fully backwards-compatible to qooxdoo 2.0. Nothing needs to be changed in the JavaScript code of your existing apps if they are based on the previous 2.0.x releases.

Bugfixes

See the corresponding bugzilla report for the issue resolved for qooxdoo 2.0.4.

qx.Desktop

  • Fixed an issue with CSS placeholders for text fields. This fix is recommended for all Firefox browsers with a version equal or higher than 19.
