qooxdoo 0.8-alpha1 (2008-05-20)

This is an early alpha release that includes and demonstrates many of the exciting feature improvements and additions of the upcoming qooxdoo 0.8.

Being a first alpha release it is neither regarded feature-complete nor API-final. One goal of this release is to provide a useful code base for first attempts at experimenting with qooxdoo 0.8 features, e.g. during the power workshops at the upcoming conferences. It is not meant for production use.

Nevertheless, feedback from you while taking a closer look and simply “playing” with this preview of qooxdoo 0.8 is highly appreciated!

Highlights

  • Feedreader application migrated and reworked
  • New default theme (demo)
  • Rounded borders and more complex decorators
  • Many widgets ported, e.g. textinput, spinner, list, toolbar, and many others
  • Layout managers reimplemented (Grid, Dock, others)
  • Cross-browser low-level DOM layer (example, more)
  • Integration into traditional web pages and page-oriented web applications
  • Animation (low-level layer)
  • and many more exciting features …
