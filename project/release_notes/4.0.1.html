qooxdoo 4.0.1 (2014-07-09)

Summary

This is a regular patch release, which includes some important bug fixes over the previous release. It is fully backwards-compatible to qooxdoo 4.0. Nothing needs to be changed in the JavaScript code of your existing apps if they are based on the qooxdoo 4.0 release.

Bugfixes

See the corresponding bugzilla query for the issues resolved for qooxdoo 4.0.1.

Details

The following sections are in alphabetic order and provide a detailed listing of issues resolved:

  • Console log breaks if className of SVG element is type of SVGAnimatedString (8432)
