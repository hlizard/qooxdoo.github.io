qooxdoo 4.0.2 (2014-12-16)

Summary

This is a regular patch release, which includes some important bug fixes over the previous release. It is fully backwards-compatible to both qooxdoo 4.0.1 and qooxdoo 4.0. Nothing needs to be changed in the JavaScript code of your existing apps if they are based on those versions.

Bugfixes

See the corresponding bugzilla query for the issues resolved for qooxdoo 4.0.2.
