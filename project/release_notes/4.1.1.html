qooxdoo 4.1.1 (2015-06-15)

Summary

This is a regular patch release, which includes some important bug fixes over the previous release. It is fully backwards-compatible to qooxdoo 4.1. Nothing needs to be changed in the JavaScript code of your existing apps if they are based on the previous version.

Bugfixes

See the corresponding bugzilla query for the issues resolved for qooxdoo 4.1.1.
